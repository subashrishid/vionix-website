# Videos Directory

Place your company intro video here as `intro.mp4`.

The video will be displayed on the home page in the "See Us in Action" section.

**Recommended specs:**
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 or 1280x720
- Duration: 30-120 seconds
- Max file size: 50MB (Netlify limit per file is 100MB)

You can change the video file path in the Admin Panel → Settings → Video File Path.
